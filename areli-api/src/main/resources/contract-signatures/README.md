# Firmas para el PDF del contrato

- **`lessor-signature.png`**: firma de **EL ARRENDADOR** (Recepciones Areli). Se muestra automáticamente en el contrato generado desde la web, encima de la línea de firma del arrendador.
- Formato recomendado: **PNG** con fondo transparente, ancho aproximado **600–1200 px** (proporción horizontal).

Para cambiar la firma, reemplaza este archivo y reinicia **areli-api** (o vuelve a desplegar el JAR).

El archivo se expone en: `GET /api/public/contract-signatures/lessor`
